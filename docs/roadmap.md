# 🛣️ FC Polotsk — Roadmap

Семь треков развития сайта. Каждый трек = независимая область, можно
делать параллельно или последовательно. Внутри трека — эпики, в эпике —
задачи с приоритетами и оценкой усилий.

## Условные обозначения

**Приоритет:**
- 🔴 **P0** — следующий спринт, высокий impact или критичный фикс
- 🟡 **P1** — в ближайшие 1-2 месяца
- ⚪ **P2** — backlog, делать когда дойдут руки

**Усилия:**
- **S** ≤ 1 день
- **M** 2-5 дней
- **L** 1-2 недели
- **XL** >2 недель (декомпозировать дальше)

## 📅 Предлагаемый порядок выполнения

### Спринт 1 (1-2 недели) — низко висящие фрукты

Эти задачи дешёвые и быстрые, но дают мгновенный value.

1. **🔴 Vercel Web Analytics + Speed Insights** (S, 30 мин на оба)
2. **🔴 sitemap.ts + robots.ts** (S, 1 час)
3. **🔴 YouTube embed в новостях** (S, 2 часа — мы это уже начинали)
4. **🔴 Inline link стили в PortableText** (S, 1 час)
5. **🔴 Sentry или Vercel Errors** (S, 30 мин)

После спринта 1 — сайт лучше для SEO, есть метрики, новости красивее.

### Спринт 2 (1-2 недели) — игроки

Решает большую боль: посетитель кликает на игрока и видит реальный профиль.

1. **🔴 Slug + bio в Sanity player** (S)
2. **🔴 Маршрут /player/[slug] с базовым layout** (M)
3. **🔴 Карточки игроков на главной кликабельны** (S)
4. **🟡 Stats engine из match.events[]** (M)
5. **🟡 Bio через PortableText** (M)

### Спринт 3 (1-2 недели) — матчи

1. **🟡 Маршрут /matches/[matchId]** (S+M)
2. **🟡 Шеринг URL отдельного матча** (S)
3. **🟡 Видео-обзор поле + рендер** (S)
4. **🟡 Фотогалерея матча** (M)

### Спринт 4 (1-2 недели) — детская школа

1. Эпики 4.1 + 4.2 целиком (M)
2. Линк в NAV (S)

### Спринт 5+ — SMM / engagement

Когда основной контент-стек закрыт, идём в шаблоны для SMM и
интеграции с соцсетями.
